## Vpeak vs Smax and Tmax vs visual t2 plots
##### Vpeak vs Smax and tmax vs visual t2
###### ( This has all the novae, V1535 Sco, V1534 Sco, V407 Cyg, V5589, V392 Per, V723 Cas, RS Oph, and U Sco.
